JapanSauna CRM 利用マニュアル

案件管理から朝のフォローアップ、活動記録まで、日々の業務で使う操作ガイド
📅 v1.0 / 2026-05-14 👤 対象：JapanSauna 営業メンバー 🔧 編集：春川 隆也

📌 はじめに

JapanSauna CRMは、これまでGoogleスプレッドシートで管理していた案件情報を一元化し、フォローアップ漏れを防ぎ、ダッシュボードで業務全体を見える化するためのシステムです。

本番URL
japansauna-crm.vercel.app
ログイン方法
Googleアカウント1クリック
対応デバイス
PC・スマホ（ブラウザ）
💡 このマニュアルの読み方：A4 2-3枚相当。最初に「朝のルーティン」と「案件対応フロー」だけ読めば日々の業務はスタートできます。残りは必要なときに参照してください。

🔐 ログイン

1
CRM URLにアクセス

ブラウザで japansauna-crm.vercel.app を開く。春川からの招待URLをブックマークしておくと便利。

2
「Googleでログイン」をクリック

会社で使っているGoogleアカウント（@japansauna.jp 等）を選択。パスワード入力は不要。

3
初回のみ：アクセス許可の同意

「JapanSauna CRMがGoogleアカウントへのアクセスを許可」画面で「許可」をクリック。次回以降は自動でログイン。

⚠️ アクセスできない場合：許可リストに登録されていないGoogleアカウントは入れません。春川に連絡してください。

🌅 朝のルーティン（5分）

毎朝、CRMを開いて以下の順に確認するだけで、その日にやるべき営業活動が分かります。

1
ヘッダーの「🔔 フォローアップ」をクリック

担当者カードから自分の名前をクリック → 自分のフォローアップだけが表示されます。

2
🔴 期限切れ から対応開始

期日を越えてしまったフォローアップ。最優先で対応。電話 or メール、何でもいいので連絡を取って活動履歴に記録する。

3
🟡 今日 を順番に対応

今日の予定アクション。期日が確定しているので、午前・午後で時間を配分して対応。

4
対応後は「活動を追加」で記録

架電・メール・商談のたびに、案件詳細から「活動を追加」。次のアクション期日も忘れず設定することで、明日以降のフォローアップに自動的に並ぶ。

💡 Slack通知：毎朝9時に営業チャンネルへ「期限切れ・今日のフォローアップ一覧」が自動投稿されます。Slackで確認してから動くのもOK。

🔄 新規問合せ→成約までの流れ

問合せ → 未通電フォロー中 → 通電済 → 商談・現調 → 見積提出 → 成約 or 失注

①「新規問合せ」が来た時

  1. ヘッダー右上の + 新規案件 をクリック
  2. 顧客名（必須）、電話、メール、地域、商材、確度（最初は E）を入力
  3. 「登録」 → 自動で案件IDが発番（例: JS-2026-05-061）

② 架電したけど繋がらなかった時

  1. 案件詳細を開く
  2. 「活動を追加」 → 種類: 📞 架電（不在/折返し）
  3. 結果に「不在・留守電」、内容に詳細
  4. 次のアクション: 「2日後に再架電」、期日: 2日後の日付を設定
  5. → 自動的にステージが「未通電フォロー中」に変更

③ 通電・商談に進んだ時

  1. 活動を追加 → 種類: ☎️ 架電（通電） や 💻 オンライン商談 や 🏠 現地調査・対面商談
  2. 会話内容や顧客の要望を 「内容」に詳しく記録
  3. 次のアクション期日を設定（「来週水曜に見積送付」など）
  4. → ステージは自動更新（通電済 → 商談・現調 等）

④ 見積を出した時

  1. 活動を追加 → 種類: 📋 見積提出
  2. 同じ案件で「関連ドキュメント」に見積書を紐付け（URL or PDFアップロード）
  3. 確度を更新（B〜A に上げる目安は 確度ガイド 参照）
  4. 次のフォロー期日（例: 3日後にフォロー架電）

⑤ 成約 or 失注になった時

成約時：パイプラインでカードを「成約」列にドラッグ。確度を「発注済」に変更。

失注時：「失注」列にドラッグ。失注理由を必ず記入（予算オーバー / 設置条件NG / 競合採用 / 計画延期 / 連絡途絶 / その他）。

🚨 失注理由の記入は超重要：月次でAIが失注パターンを分析し、商品開発・営業改善の材料になります。空欄だと分析できません。

🎯 確度の判定基準

確度の判定はチーム全員で揃えることが重要。下記の基準を満たしたら、その確度を付けてください。

確度要約判定基準（3つ以上満たす）想定LT
A 発注ほぼ確実 意思決定者と話済・現調済・見積合意済・発注時期確定・競合終了 1ヶ月以内
B 見積OK・予算OK 見積提出済・設置場所確認済・予算範囲OK・競合検討段階 1〜2ヶ月
C 商談・現調済 商談or現地調査済・仕様詰め中・予算未確定だが意向あり 2〜4ヶ月
D 興味あり程度 通電済・一次商談済・興味はあるが具体化はこれから 半年〜不明
E 問合せのみ 問い合わせのみ・情報収集段階・数年先案件 1年以上
📖 詳細はCRM内のヘルプページに：CRMにログイン後、フォーム編集画面の確度欄に「確度の業務基準を見る →」リンクあり。

📐 運用ルール

入力ルール

  • 新規問合せは1営業日以内に登録（フォーム自動連携が動けば自動）
  • 架電・メール・商談のたびに必ず活動履歴を記録。「言った言わない」を防ぐ。
  • 次のアクション期日を必ず設定。空欄だとフォローアップ画面に出てこない＝放置される。
  • 確度の変更は判定基準に従う。「Aを付けるなら3つ以上満たす」を厳守。
  • 失注時は理由を必ず記入。月次AI分析の精度が決まる。

フォローアップの目安

状況次のアクション期日
架電不在1〜2日後に再架電
メール送信後（返信待ち）3日後にフォロー
商談後（次回未定）3日〜1週間後に近況確認
見積提出後（検討中）1週間後にフォロー架電
長期検討（D / E）1〜3ヶ月後に状況確認

禁止事項・注意点

  • 個人情報を SNS や外部メモアプリに転記しない。CRM内で完結させる。
  • 確度の感覚的な上げ下げをしない。判定基準ベースで。
  • 同じ顧客の重複登録に注意。先に検索して確認。

❓ よくある質問

質問回答
ログインできなくなった 許可リストから外れていないか 春川 に確認。または、シークレットウィンドウで再ログイン。
間違って入力した。元に戻したい 案件詳細の「編集」から修正可能。活動履歴は削除アイコンで削除可。
