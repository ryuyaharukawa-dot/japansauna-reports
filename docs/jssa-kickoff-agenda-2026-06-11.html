BACKGROUND

背景資料（参考）

CONTEXT 01

業界の現状：安全基準が存在しない

  • サウナタイガー事案後、消防・保健所が全国で巡回強化
  • しかし、個人宅・商業施設キャビンへの統一安全基準が未整備
  • 消防庁は2026年3月に省令改正（テント・バレル型対象）したが、キャビン型は空白
CONTEXT 02

既存協会との差別化ポジション

  • 日本サウナ・スパ協会 → 公衆浴場・大型スパが主対象
  • 日本サウナ学会 → 感染症対策・健康科学が軸
  • JSSA：個人宅・ホテル・ヴィラのキャビン設置・運用基準に特化
CONTEXT 03

JSSAがJapanSaunaにもたらす価値

  • 「基準を作った会社」としてのブランド優位性
  • キャビン販売時に「認定準拠製品」として訴求可能
  • The・のサウナ = 第一号認定施設としてPR
CONTEXT 04

理事構成の考え方

  • 代表理事：中立な第三者（消防庁OB・大学教授等）
  • 理事枠：消防・防災 / 建築・設備 / 医療・健康（塩谷先生） / 施設オーナー
  • JapanSaunaは理事＋事務局として実務を担う
塩谷先生にご参加いただくことで、医療・健康の専門性とサウナ業界での信頼・認知を協会に持ち込むことができます。既存のサウナ学会活動とは領域（設備安全基準）が異なるため、利益相反も生じません。
REQUEST

塩谷先生へのご依頼

依頼 01

医療・健康分野の理事として参加いただきたい

JSSAの理事として、医療・健康の専門家の視点から安全基準策定にアドバイスをいただきたいと考えています。

  • 安全基準の骨子策定における医学的知見の提供
  • 年2〜3回程度の理事会参加（オンライン可）
  • 対外的な信頼担保のためのお名前・プロフィール掲載
  • （任意）プレスリリース・HP上でのコメント
日常業務への大きな工数は想定していません。まずは設立時の理事として名義参加いただく形でも構いません。
依頼 02

代表理事候補をご紹介いただきたい

協会の中立性・公益性を担保するため、JapanSaunaは代表理事に就かず、第三者の方にお願いしたいと考えています。

以下のプロファイルに近い方で、お心当たりがあればぜひご紹介ください。

  • 消防庁・総務省OB（行政との信頼関係が即できる）
  • 建築・設備安全の大学教授（日本建築学会・防災工学系）
  • 製品安全・リスクマネジメントの専門家（製品安全協会・損保出身など）
